# NaSol-AI-Server!

# 재무 정보 분석 가이드

## 📂 대주제 : 재무 문서 기반 개인 자산 분석 AI

<aside>
💡

PDF, 이미지, 스캔문서, 명세서, 영수증 등의 **재무 정보 관련 문서를 자동으로 분석**하고, **추가적으로 검색, 질의, 요약**까지 가능한 서비스

- 문서를 분석하고
- 정보를 요약하고
- 특정 질문에 답변
</aside>

## 📌 **주요 기능 정리**

1️⃣ **문서 업로드 기반 분석(핵심)**

- 급여명세서, 원천징수영수증 → 소득 정보 추출
- 카드 명세서, 고정지출 문서 → 지출 정보 추출
- 주요 문서 분석 대상 타입
    - 영수증(사진/스캔) - 이미지 기반 문서 포함
    - 거래 명세서
    - 계약서
    - 공문/보고서 (PDF)
    - 표/테이블 포함 문서
    - 원천징수영수증
    - 급여명세서
    - 홈택스 연말정산 간소화 자료
- 문서 종류에 따라 다음 항목을 추출 → JSON 형태로 제공(추후 논의)
    - 제목/일자/발신자/수신자
    - 총 금액, 품목 (영수증)
    - 조항별 구조 (계약서)
    - 표/테이블 데이터
    - 문서 메타데이터
    - 지출별 카테고리 (EX; 식비, 카페, 의류비)
        - 소득별 지출 비중 가이드
        - 신용/체크/현금영수증 사용액 구분
        - 국민연금·건강보험·고용보험 납부액
        - 연금저축/IRP 납입액
        - 의료비, 교육비, 기부금 등

2️⃣ **재무 분석 + 가이드 생성**

- 자연어 질의응답 가능
- 현재 소득/지출 기반 자산 분배 추천
    - 사용자의 성향 고려(위험성향/안전성향)
    - 사용자의 목표기간 고려(단기/중기/장기)
    - 총평 요약 리포트 - 그래프 등 시각화 자료 조회 및 다운로드
        - 월별 지출 히트맵
        - 카테고리 비중 파이차트
        - 연말정산 공제 예상 금액 그래프
        - 미래 자산 시뮬레이션
- 세액공제 가능 항목 가이드(연말정산 시즌 아이디어)
    - 세액공제 가능 항목 체크 리스트
        - 공제 가능한 항목 자동 식별
        - 누락된 공제 항목 추천
    - 세액공제 예상 금액 산출
    - 개인 맞춤형 절세 전략 제안
- 사용자 재무상태 지속 모니터링 및 주기적인 가이드 제공
    - 목표 금액 설정 및 달성까지의 가이드
    - 사용자 지정 주기별 자동 분석 (예) 매주/매달/매년
    - 이상 지출 감지 및 경고
    - 투자 비중 자동 조정 안내

3️⃣ **옵션 기능**

- 문서 없이도 대략적 입력만으로 간단한 가이드 제공
    - OCR: 임의의 이미지 파일에서 데이터 인식 후 자동 입력
    - 사용자 동의 하에 이메일에서 문서 자동으로 불러오기
    - 지정된 폴더에서 문서 자동 탐지 기능 (예) 다운로드 폴더
    - 통장(자산) 정보 불러오기
    - 금융앱 SMS 자동 파싱
- 회사별 제출 포맷 (반)자동 매핑

‼️데이터·법적 가이드라인 문구 보강

⇒ 사용자 동의(수집·처리), 보관기간, 삭제 요청 절차, 비식별화 옵션 명시 필요

**+** 정확성 확보 모듈 추가(?)

⇒ 계산 검증(합계/부가세/공제 합계 검증) → 불일치 시 ‘검증 필요’ 플래그

샘플 기반 골든셋(50~200건)으로 Extraction Accuracy 측정하면 되나

---

- (시간 남으면) 비슷한 소득/지출 패턴 가진 사람들의 자산 분배 비교 서비스
- 급여명세서, 원천징수영수증 등의 소득 정보를 입력(업로드) 할 수 있다.
- 고정지출, 카드 명세서 등의 지출 정보를 입력(업로드)할 수 있다.
- 입력 받은 문서(PDF 파일, 폼 데이터 등)을 AI를 통해 분석한다.
- 현재 소득/지출 등의 정보를 통해 자산 분배 가이드를 얻을 수 있는 AI를 목표로 한다.
- 대략적인 정보(폼 데이터 입력)를 통해 가이드를 받거나, 문서 업로드를 통해 상세/정밀한 가이드를 받거나 사용자가 선택할 수 있게 한다.

|기능	|난이도	|필요 기술|
|:---|:---:|:---:|
|OCR + 문서 분할	|중	|Tesseract, Google Document AI, LayoutLM 등|
|세액공제 항목 자동 분류|	중상|	룰 기반 + LLM 조합|
|누락 항목 추천	|중상	|패턴 분석 + LLM reasoning|
|세액공제 계산	|낮음	|규정 기반 알고리즘|
|개인 맞춤 리포트 생성	|중	|LLM 템플릿|
|IRP/소득 시뮬레이션|	낮음	|단순 수식|
